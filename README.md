# vite项目

## 1. 安装使用

```shell
git clone https://github.com/huoyuhao/vue-ts.git
npm install
npm run dev
# 打包
npm run build
# 修复前端规范
npm run lint
```

## 2. 项目环境

### 2.1 环境搭建

+ git项目创建
+ vite初始化
+ vite基础配置管理
+ vite环境变量ENV引入
+ vite打包配置
+ 测试环境请求代理、跨域配置

```shell
npm init @vitejs/app admin
# 选择vue -> 选择 vue-ts
cd admin
npm install
npm run dev
```
